Términos de servicio

Última actualización: 24/02/2026

Estos términos regulan el uso de Simplify. Al utilizar el servicio, aceptas estas condiciones.

1. Uso permitido

  • Puedes usar Simplify para resumir, simplificar o transformar textos propios o con permiso de uso.
  • No está permitido usar el servicio para actividades ilícitas o que vulneren derechos de terceros.

2. Disponibilidad y cambios

El servicio puede cambiar, pausarse o actualizarse para mejorar su estabilidad y seguridad.

3. Pagos y créditos

Los pagos y suscripciones se gestionan con terceros proveedores. Revisa sus políticas antes de completar una compra.

4. Contacto

Para consultas sobre estos términos, visita la página de contacto.
